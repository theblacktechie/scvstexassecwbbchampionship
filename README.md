# South Carolina vs. Texas · SEC Championship 2026

**The Work Continues.** An interactive data visualization covering South Carolina's SEC Championship loss to Texas on March 8, 2026.

Part of an ongoing series of sports data visualizations by Kris E. Smith.

---

## Overview

Texas won the 2026 SEC Championship 78–61, powered by a dominant first quarter. South Carolina trailed 27–12 after one period but fought back competitively in the final three quarters. This viz tells that full story through real game data.

Built in the same visual language as the [SC vs. LSU SEC Tournament visualization](https://theblacktechie.github.io/scvslsu2026sectourney/), using Inter font and the Beyond Measure editorial design system.

---

## What's Inside

**Score Band**
Final score with quarter-by-quarter splits for both teams. Click the Texas 78 to trigger a confetti celebration.

**Key Numbers**
Four context stats that frame the game: the Q1 point deficit, SC's seventh consecutive SEC title game appearance, Texas categories won and SC's category won.

**Team Identity**
Season defining stats for both South Carolina and Texas. Toggle between offensive and defensive profiles using the OFF and DEF buttons on each card.

**Head to Head Stats**
Animated bar chart comparing both teams across eight game categories with edge badges. Toggle between This Game and Season Avg to see how the matchup compares to each team's season performance.

**Key Performers**
Six player cards covering the top performers from both teams, with game stats and shooting detail.

**The Verdict**
Editorial closing section framing the result and pointing forward to the NCAA Tournament.
